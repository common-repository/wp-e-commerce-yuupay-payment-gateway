== Description ==

YuuPay payment for WordPress with WP E-Commerce plugin allows you to process payments using the YuuPay payment gateway.
If you found any questions please contact us at info@yuupay.com. 

== Installation ==

1. Upload `yuupay.php` and `yuupay_lib` to the `/wp-content/plugins/wp-e-commerce/merchants` directory
2. Make sure there are 1 new file ('yuupay.php') and 1 folder ('yuupay_lib').
3. Configure you YuuPay at WordPress admin -> (WP E-commerce) Store Settings -> Payment Options
4. Check YuuPay on the left List and click Update.
5. Select YuuPay on the right list, and there will be a form to fill :
	5.1 Display name : Name that will be displayed in the payment/checkout options.
	5.2 YuuPay MID : Your YuuPay Merchant ID.
	5.3 YuuPay Server Host : Don't change the default value. 
	5.4 Enable 4G Secure : Additional Security from 4G Secure.
6. Click update to save the changes.
